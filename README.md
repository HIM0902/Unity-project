# Dead Silence

Dead Silence is a Unity 6 zombie survival game project created for CS 4332. The game focuses on stealth, sound-based zombie AI, and survival gameplay.

## Requirements
Before opening the project, make sure you have:

- Unity Hub installed
- Unity 6 installed
- Git installed
- A GitHub account with access to this repository

## Recommended Unity Version
Use the Unity version listed in the project folder or GitHub files.  
If needed, install the correct version through Unity Hub before opening the project.

## How to Import the Project from GitHub into Unity

### Option 1: Clone with GitHub Desktop
1. Open **GitHub Desktop**.
2. Sign in to your GitHub account.
3. Click **File > Clone Repository**.
4. Select this repository.
5. Choose a local folder to save it.
6. Click **Clone**.

### Option 2: Clone with Git Command Line
1. Open a terminal or command prompt.
2. Navigate to the folder where you want the project saved.
3. Run:

git clone https://github.com/HIM0902/Unity-project.git
